# Project Title: Billing System Database

This project is a simple Billing System database designed using SQL.
It includes customer records, product details, orders, order items, and payments.

## The project demonstrates your ability to:

1. Design database schemas

2. Create relational tables

3. Implement foreign keys

4. Insert data

5. Write queries

6. Understand ER diagrams

# Project Overview

This SQL project simulates a small shop's billing management system.
It helps track:

## Customer details

1. List of products

2. Orders placed

3. Items inside each order

4. Payment transactions

 # Database Design
## Entities Used:

- Customers

- Products

- Orders

- Order Items

- Payments

# Relationships:

- A customer can place multiple orders

- An order can have multiple order items

- Each order has one payment

- Products are linked to order items

# SQL Features Covered

## This project includes:

- CREATE TABLE

- PRIMARY KEY, FOREIGN KEY

- INSERT INTO

- Relational joins

- Basic queries

- Payment tracking

- Order item total calculation
